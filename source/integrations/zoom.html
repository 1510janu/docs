Zoom Plugin ================================ 1. The System Console settings live in System Console > Plugins > Zoom. This is where the admin needs to go configure Zoom for Mattermost. 2. There are steps where you need to change a few settings in Zoom. 3. Once these steps are done, activate the plugin at System Console > Plugins > Management. 4. Once activated, you will see a video icon in the channel header. There are three different actions you can take (start meeting now/start meeting with topic/share meeting). Setting for overriding usernames and icon URLs need to be set to true. API Key ......................................... The API Key generated by Zoom, used to create meetings and pull user data. To generate an API Key: 1. Go to https://developer.zoom.us/me/ and login. 2. Switch to the API tab and click Enable, if not already enabled. 3. Switch back to the API tab and copy the API Key. 4. Paste the API Key into the field above. API Secret ......................................... The API Secret generated by Zoom, used to create meetings and pull used data. To generate an API Secret: 1. Go to https://developer/zoom.us/me/ and login. 2. Switch to the API tab and click Enable, if not already enabled. 3. Switch back to the API tab and copy the API Secret. 4. Paste the API Secret into the field above. Webhook Secret ......................................... The secret used to authenticate the webhook to Mattermost. (Button text) Regenerates the secret for the webhook URL endpoint. Regenrating the secret invalidates your existing Zoom webhook. Footer ......................................... Use this webhook URL to set up the Zoom webhook. See documentation to learn more.
